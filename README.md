# clk
<img width="847" height="549" alt="image" src="https://github.com/user-attachments/assets/262ba636-6dc6-49a7-b3df-934f51bf239d" />

clk is a open source chess clock that is comparable to closed source alternative

### why clk?

if you google open source chess clock, here's the result you'll get (SOURCE: https://github.com/zabackary/open-chess-clock). it's a cool project, but i wanted to make something better. something that looks like an industry giant made it. thats why i build clk.
